# SilverKaki
SilverKaki is an interdisciplinary digital solution created for PROID Assignment 2: Group Comprehensive Solution. The app serves as a companion for seniors aged 60+ to navigate NHG Community Care and Active Ageing Centre (AAC) programmes.

Key Features:
- Front-End Health Overview: Simplified display of blood pressure records and basic health indicators.
- Activity Matching: Intelligent program suggestions based on mobility levels (Sit/Stand/Walk) and fall risk considerations.
- "Fetch Me" Transport Support: A wayfinding feature allowing seniors to request transport assistance from AAC staff.
- Community Support: A simplified forum for peer interaction and health discussion.
- Incentive System: Visual progress tracking and NTUC-linked rewards to encourage active participation.
